# Spatial Transcriptomics Access Guide

The datasets can be found deposited in the Gene Expression Omnibus (GEO) database under accession code (GSEhere). The count matrix, tissue_positions.parquet, scalefactors_json.json, and image files can be input into Python or R for analysis, such as with Scanpy, Squidpy, or Seurat. Treatment information can be added as metadata to the AnnData or Seurat object, with our labelling for the 8 µm bin resolution deposited as .csv files in "Spatial replicate timepoint csv files".

For intuitive or quicker analyses, you can use the Loupe Browser from 10x Genomics:

The following link can be used to access a download center for Loupe Browser or LoupeR packages, system requirements, and tutorials. 
https://www.10xgenomics.com/support/software/loupe-browser/latest
1.	Download the latest version of Loupe Browser found on 10X Genomics website. There is a version for Windows or MacOS. You can use the link below. 
https://www.10xgenomics.com/support/software/loupe-browser/downloads

2.	Download the dataset (8 µm bin resolution, cloupe.cloupe file) from GEO GSEhere

3.	To separate each slide by treatment, the brush tool or freehand selection along the top right of the Loupe Browser can be used to separate the bins manually. Alternatively, the bins corresponding to each treatment are listed in .csv files located under "Spatial replicate timepoint csv files" which can be uploaded to the Loupe Browser by selecting the Upload icon along the bottom left, to the right of "Create a New Group".

4.	Once you have downloaded Loupe Browser and the dataset, you can open the dataset and start becoming familiar with how to use Loupe Browser to explore the data. The file may take several minutes to open, depending upon the size.

    The link below provides a comprehensive walkthrough of most of the features and analysis tools.
-	Please note that the spot deconvolution feature does not work on Visium HD data. 
https://www.10xgenomics.com/support/software/loupe-browser/latest/tutorials/introduction/lb-navigation-for-spatial

5.	The link below provides a slightly more in-depth description on analysis using unsupervised clustering, mapping “features” (genes of interest) on the image, mapping specific cell types, filtering barcodes, running DGE, and using the co-expression feature. 
https://www.10xgenomics.com/support/software/loupe-browser/latest/tutorials/assay-analysis/lb-hd-spatial-gene-expression

With "Clusters" selected along the left, you can select between the Space Ranger generated Graph-based or K-means (2-10) clusters to visualize on the tissue. 

"Features" allows for different visualizations of select genes or gene lists, which can be manually entered or uploaded from a csv file along the bottom left. Co-expression within "Features" or "Advanced Selection" along the left can be used to select bins with certain expression levels of specified genes.
